ИГРА "МОРСКОЙ БОЙ"

Задачи:

1. доработать метод field.placeShips() таким образом, чтобы корабли расставлялись случайным образом с соблюдением правил игры "Морской бой"

2. доработать метод field.placeShips(). В случае выстрела по координате, по которой выстрел уже производился (море или корабль), выводить в лог программы предупреждение о некорректности работы алгоритма "стрелка"

3. доработать метод field.isGameOver(). В случае, если подбиты все корабли, он должен возвращать True

4. доработать метод field.showGameOver(). Он должен показывать красивое сообщение, что игра завершена. Желательно еще показывать количество произведенных выстрелов.

5. доработать метод belovFleet.shoot(). Он не должен повторно стрелять туда, куда уже стрелял.

6. доработать основную программу main, чтобы можно было приостановить и продолжить работу алгоритма нажатием на пробел.

7. реализовать свой модуль, аналогичный belovFleet с реализацией собственного алгоритма ведения боя. Например, попав по кораблю, стараться сразу потопить его полностью, и больше не стрелять по клеткам рядом с ним.

8. написать новую программу main2 и модуль field2, позволяющие играть компьютеру против компьютера.
